Below is a list of Elements

  • h1 , h2 , h3 , h4 , h5 , h6 elements are header elements
  • p element is for paragraphs
  • em element is Emphasized Text
  • strong element is Bold Text
  • br element breaks the text into another line
    like so
  • ol element stands for ordered list
  • ul element stands for unordered list
  • li element stands for list which can be added under ul and ol
  • video element is exactly what is sounds like it does
  • img element means it adds images
  • head element is used to section off the code's header
  • title element is used for the title tab of the HTML page
  • div element is used to divide content into sections
  • span element is used for CSS to address specific text
  • a element is used to open other webpages
